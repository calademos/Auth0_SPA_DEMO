Setup

Edit the config below with your Auth0 values. This file can be hosted statically (e.g., GitHub Pages). No backend required.

// Replace these placeholders
const config = {
  domain: "dev-zuk2nsc6i50ge88q.us.auth0.com", // e.g. dev-abc123.us.auth0.com
  clientId: "Osyw9ZHWpkYIcpHDkgtmBo2eMmVLvUN8",
  audience: "", // optional; set if you need an API access token
  redirectUri: window.location.origin + window.location.pathname.replace(/\/?index\.html?$/, ''), // must be on Allowed Callback URLs
  cacheLocation: "memory", // or "localstorage" (for legacy flows)
  useRefreshTokens: true // set to true if enabled in Auth0 App settings (Refresh Token Rotation)
};

In your Auth0 Application (SPA): add window.location.origin to Allowed Web Origins and window.location.origin + '/' to Allowed Callback/Logout URLs.
